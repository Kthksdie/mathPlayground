## Structure

This repository contains two main implementations:

### [`/cs`](./cs)
The C# .NET workspace containing prototypes and experiments.

#### Projects
-   **`Numberality.Con`**: The original console prototype containing initial experiments and "Trails" (versions) of the algorithm.
-   **`TestingNumbers`**: An extensive test suite for exploring number properties, sequences, and advanced factorization patterns.


### [`/python`](./python)
The python workspace containing prototypes and experiments.

#### Projects
-   **`reverseMultiply.py`**: A CLI-based port of the core logic from C# [`/cs/Numberality.Con/Components/ReverseMultiply.cs`](./cs/Numberality.Con/Components/ReverseMultiply.cs)
